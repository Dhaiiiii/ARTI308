## Dataset Description
The dataset used in this lab is a tabular dataset based on the Titanic passengers data.
It contains information about passengers such as age, fare, sex, class, and other encoded features.
The dataset has 1309 rows and 28 columns.

## Machine Learning Problem
This is a classification problem.
The target variable is the passenger survival status (2urvived).
The goal is to predict whether a passenger survived based on the available features.
